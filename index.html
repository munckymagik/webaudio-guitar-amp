Web Audio Guitar Amp

This is a simple demo I made to experiment with processing a guitar signal using the Web Audio API. It was made for the Scotland JS 2015 Web Audio Hack Day.

It's known to work on Google Chrome 42.0.2311.135.

Connect your guitar then (gradually) turn up the volume control in the Amplifier module. Enjoy!
